# Content Intake Folder

Use this folder to provide source material before publishing final content.

## Required First

1. Fill `business-profile.md` with exact business details.
2. Add one file per project in `projects/` based on the template.
3. Place original photos in `photos/` using clear names.
4. Add city-specific notes in `cities/`.

## Naming Conventions

- Photos: `city-service-short-description-01.jpg`
- Project files: `YYYY-MM-project-slug.md`
- City notes: `city-slug.md` (example: `chattanooga-tn.md`)

## Quality Guidelines

- Use real project details, not generic text.
- Include measurable outcomes and constraints.
- Include neighborhood references where relevant.
- Keep claims accurate and defensible.
